## User Flows

### Creating a New Archive
1. User taps **[+]** button on home screen
2. Navigates to "Create Archive" screen
3. Enters archive name
4. Enters password (minimum 6 characters)
5. Confirms password
6. Taps "Create Archive"
7. Returns to home screen with new archive visible

### Opening an Archive
1. User taps on an archive card
2. Password dialog appears
3. User enters password
4. If correct: archive detail screen opens
5. If incorrect: error message shown, dialog remains

### Adding Files
1. User opens an archive
2. Taps **[+]** button
3. System file picker opens
4. User selects one or more files
5. Loading indicator shown
6. Files added to archive
7. File list updates with new files

### Removing Files
1. User opens an archive
2. Taps **[ğŸ—‘]** button next to a file
3. Confirmation dialog appears
4. User confirms deletion
5. File removed from archive
6. File list updates

### Deleting Archives
1. User taps **[â‹®]** menu on archive card
2. Selects "Delete"
3. Confirmation dialog appears
4. User confirms deletion
5. Archive and all its files deleted
6. Home screen updates
